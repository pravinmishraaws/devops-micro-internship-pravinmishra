#### Question 2 — What is AWS Free Tier, and how long does it last?

The AWS Free Tier is a program that allows users to access and test various cloud services for free, as long as they stay within specified monthly usage limits. It is split into three main categories:

**12-Month Free Services** — Core resources like EC2, RDS, and S3 are free for exactly one year from signup. (Note: Accounts created after July 2025 get 6 months + $200 in starter credits instead).

**Always Free Services** — Lightweight services like Lambda (up to 1 million requests/month) never expire and reset monthly throughout the account lifetime.

**Short-Term Trials** — Service-specific trials lasting 30-90 days that begin when each service is activated for the first time.

This tiered approach lets account users to experiment with AWS cost-free while learning cloud basics before committing to paid services.

---

#### Question 3 — Name three AWS Free Tier services and their free usage limits.

To address this for an assignment or technical documentation, here are three key **AWS** Free Tier services and their specific monthly usage limits:
- Amazon **EC2** (Elastic Compute Cloud): Provides **750** hours per month of a t2.micro or t3.micro virtual machine instance (enough to run one virtual server continuously 24/7). This offer is available for the first 12 months of a new account. 
- Amazon S3 (Simple Storage Service): Provides 5 GB of standard object storage, along with 20,**000** **GET** requests and 2,**000** **PUT** requests per month. Like **EC2**, this is valid for the first 12 months. 
- **AWS** Lambda: Provides 1 million free requests and up to **400**,**000** GB-seconds of compute time per month. This is an Always Free service, meaning the limit resets monthly and does not expire after the first year.
