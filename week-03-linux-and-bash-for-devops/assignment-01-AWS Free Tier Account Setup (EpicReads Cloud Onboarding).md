# Task 1 — Understanding AWS & Free Tier

## Goal

Demonstrate understanding of AWS basics and Free Tier usage by answering the following questions in your own words (3–4 lines each).

### Answers

#### Question 1 — What is an AWS account, and why do you need it at this stage?

An AWS account is the container for everything done in AWS. It is a private space within AWS's infrastructure, tied to a specific email, with its own billing, its own resources (such as EC2 instances, S3 buckets, and VPCs), and its own security boundary.

I needed an AWS account at this stage because I wanted to deploy a React application on a cloud server, and an AWS EC2 instance would serve this purpose.

---

#### Question 2 — What is AWS Free Tier, and how long does it last?

AWS Free Tier is a feature that allows new customers to try out AWS services at no cost. When creating a new account, you can choose the Free plan, which comes with $100 in credits (up to $200 after completing onboarding activities). 

The Free plan lasts up to 6 months, or until the credits run out whichever comes first with no charges incurred. AWS also offers 30+ "Always Free" services with permanent monthly usage limits that never expire, regardless of the account's age.

---

#### Question 3 — Name three AWS Free Tier services and their free usage limits.

**Free Trials** – Use some AWS services for a short period until the free trial expires.

**12 Months Free** – Enjoy free usage of certain AWS services for 12 months, starting from the date of account creation.

**Always Free** – Some AWS services offer a certain amount of usage that is always free. For example, AWS Lambda offers 1 million free invocations per month, after which charges are incurred.
